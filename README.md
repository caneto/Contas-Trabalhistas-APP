# 📊 Contas Trabalhistas

Aplicativo desenvolvido para auxiliar no cálculo e controle de **contas trabalhistas**, oferecendo praticidade, rapidez e precisão para profissionais da área jurídica, contábil e também para trabalhadores que desejam entender melhor seus direitos.

---

## 🚀 Sobre o Projeto

O **Contas Trabalhistas** é um aplicativo mobile criado com foco em facilitar o dia a dia no cálculo de verbas trabalhistas, eliminando a complexidade de planilhas manuais e reduzindo erros comuns.

Com uma interface simples e intuitiva, o app permite realizar cálculos de forma rápida, organizada e confiável.

---

## ✨ Funcionalidades

* 📅 Cálculo de **Rescisão Trabalhista**
* 💰 Cálculo de **Férias + 1/3 Constitucional**
* 🧾 Cálculo de **13º Salário**
* ⏱️ Controle de tempo de serviço
* 📊 Simulações completas de valores
* 📱 Interface prática e fácil de usar
* 🔒 Dados processados localmente (mais segurança)
